# 單選題
1. 项目的生命周期首先要考虑的是什么？
```
A： 方案设计
B： 上线/交付
C： 用户需求
D： 人肉运维
正确答案：C
```

2. 金牛产品是属于那种产品？
```
A： 高市场占有率，高销售增长率
B： 低市场占有率，低销售增长率
C： 低市场占有率，高销售增长率
D： 高市场占有率，低销售增长率

```

```
A： 预测型
B： 增量型
C： 迭代型
D： 敏捷型
正确答案：B
```

4. 下列哪一项不属于项目的特征？

```
A： 稳定性
B： 临时性
C： 独特性
D： 不确定性
正确答案：A
```

5. 传统模式中企业是按照什么来划分部门的？
