# 單選題
1. 下列选项中可能获得发明专利的是
```
A： 一种计算圆周率的方法
B： 一种新型电脑芯片
C： 一幢造型别致的建筑物
D： 一种扑克牌的新玩法
正确答案：B
```

2. 我国专利法规定发明专利的保护期为
```
A： 10年
B： 15年
C： 20年
D： 30年
正确答案：C
```

3. 专利的“三性”不包含哪种？
```
A： 新颖性
B： 实用性
C： 秘密性
D： 创造性
正确答案：C
```
