1. 全球最低税负制下，辖区有效税率低于多少时需要补足课税？
```
A： 25%
B： 20%
C： 15%
D： 10%
正确答案：C
```

2. 以下何种情况构成间接转让在中国的财产？
```
A： A股东转让开曼公司，这家开曼公司仅100%持有一家位于中国的企业
B： A股东转让位于中国的一家公司

D： A股东持有开曼公司，开曼公司转让其持有的中国公司股权
正确答案：A
```

C： 是；中国及开曼
