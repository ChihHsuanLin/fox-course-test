# 單選題
1. 要达到企业组织与人才盘点的目标，此项作业的基本思维是
```
A： 以始为终
B： 以终为始
C： 顺势而为
D： 瞻前顾后
正确答案：B
```

2. 在人才盘点中的主要角色有所不同，哪一个才是人才盘点的真正owner
```
A： 人力资源
B： 组织文化
C： 企业人才
D： 业务领导者
正确答案：D
```

3. 以下哪一项不是决定人才盘点的成功要素
```
A： 主观的意见
B： 业务策略
C： 高层投入与承诺
D： 与人力资源整合的一致性
正确答案：A
```

4. 以下哪一项是人才较为不关注的事
```
A： 适切的薪酬
B： 上司是否关注我
C： 工作餐的美味与否
D： 工作的前景与可发挥性
正确答案：C
```

5. 下列哪一项不是盘点讨论时的重要规则
```
A： 保密
B： 心态开放
C： 局部
D： 实事求是
E： 多视角
正确答案：C
```
