# 單選題
1. 以下哪项不属于结果导向思维的表现形式
```
A： What（要做什么事）
B： How（要怎么做）
C： Who（谁去做）
D： Why（为什么这么做）
正确答案：C
```

2. 企业管理的结果导向是为了实现什么？
```
A： 实现过程最优化
B： 实现企业效益的最大化
C： 实现周期最短化
D： 实现成本最小化
正确答案：B
```

3. 以下哪一项不属于结果的三要素
```
A： 时效性
B： 实效性
C： 可视性
D： 可实现性
正确答案：D
```

4. 以下哪项不属于结果导向的工作难点
```
A： 工作交流是难以进行的
B： 工作内容是交叉复杂的
C： 工作限制是动态变相的
D： 工作难度是逐渐上升的
正确答案：A
```

5. 以下哪一项不属于结果导向的思维
```
A： 达到什么目的
B： 取得什么效果
C： 需要什么样的过程
D： 结果的责任人是谁
正确答案：D
```
