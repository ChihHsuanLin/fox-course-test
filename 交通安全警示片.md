# 單選題
1. 驾驶机动车时，遇前方停驶的油料运输车起火冒烟，以下正确的是（ ）
```
A： 加速通过
B： 停车上前查看
C： 尽量远离并拨打报警电话
D： 停车后围观
正确答案：C
```

2. 同车道行驶的机动车，与对面有（ ）可能的，不允许超车
```
A： 会车
B： 避让
C： 碰撞
正确答案：A
```

3. 下列不属于交通信号的是（ ）
```
A： 交通信号灯
B： 交通信号板
C： 交通警察的指挥
正确答案：B
```

4. 机动车发生碰撞时座椅安全带的主要作用是（ ）
```
A： 保护驾乘人颈部
B： 保护驾乘人胸部
C： 减轻驾乘人员伤害
D： 保护驾乘人员腰部
正确答案：C
```

5. 不得连续驾驶机动车超过（ ）小时未停车休息或停车休息少于（ ）分钟
```
A： 4/20
B： 8/30
C： 2/15
正确答案：A
```

6. 机动车行驶中遇雾、雨、雪、沙尘、冰雹等能见度在50米以内时，应当（ ）让行
```
A： 加速通过
B： 减速
C： 停车
正确答案：B
```

7. 醉酒驾驶机动车在道路上行驶会有怎么样的惩罚（ ）
```
A： 处两年以下有期徒刑
B： 处拘役并处罚金
C： 处两年以上徒刑
正确答案：B
```

8. 驾驶摩托车转弯时，如旁边有货车或拖挂车等大型车辆同时转弯时应（ ）
```
A： 与大型车辆保持1.5米以上的横向车距
B： 与大型车辆保持1-1.5米的横向车距
C： 与大型车辆保持0.5-1米的横向车距
正确答案：A
```

9. 驾车系安全带在遭遇紧急制动或发生碰撞时会避免（ ）
```
A： 撞击挡风玻璃
B： 被甩出车外
C： 造成胸部损伤
D： 以上都对
正确答案：D
```

10. 以下哪些行为不属于不良的安全行为（ ）
```
A： 闯红灯
B： 路口抢行
C： 与前车保持安全距离
正确答案：C
```
