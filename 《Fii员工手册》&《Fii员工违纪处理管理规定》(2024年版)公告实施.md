# 單選題
1. 上班时间，做与工作无关之事宜，浏览与工作无关网页、电子书、视频、小说、报刊杂志、上网聊天、进餐等，经劝导不改正或屡犯者（）
```
A： 记警告一次
B： 记小过一次
C： 记大过一次
正确答案：A
```

2. 年休假以（）小时为最小休假单位进行休假，单次休假原则上不得超过（）天；
```
A： 1小时；3天
B： 1.5小时；5天
C： 1小时；10天
正确答案：C
```

3. 拾获他人财物不归还，经教育、劝说拒不如实交代或拒不返还财物者，或其他情节严重者（）
```
A： 记小过一次
B： 记大过一次
C： 开除
正确答案：B
```

4. 在厂区（含厂区门岗外门岗管理区域）或厂区公寓/宿舍区域打架，打架斗殴、或先动手打人、或造成人员伤亡者（）
```
A： 记警告一次
B： 记小过一次或大过一次
C： 开除
正确答案：C
```

5. 将本人或他人厂牌或识别证等证件(含假借贵宾卡等)借与非本公司人员使用，或袒护非本公司人员混入厂区或厂区公寓/宿舍，在进门岗时被查获者，或未造成严重后果者（）
```
A： 记小过一次
B： 记大过一次
C： 开除
正确答案：B
```

6. 在厂区或厂区公寓/宿舍区吸烟、乱扔烟头，在其他非高危险禁烟区(含厂区公寓/宿舍、运动场、公共区域等），经提醒拒不改正，或态度不好者（）
```
A： 记警告一次
B： 记小过一次
C： 记大过一次
正确答案：C
```

7. 滥用职权、营私舞弊，为本人或他人（包括厂商）谋取不正当利益者（）
```
A： 记小过一次
B： 记大过一次
C： 开除
正确答案：C
```

8. 未经许可，私自携带SIM卡或具有通讯、录音、摄影/像、可移动式存储、复制等功能的设备（如：U盘、移动音频播放器、存储卡、带摄像头手机、智能型手机、智能型手表、平板电脑等）进出生产区域、研发部门或其他重点管制区域(一经发现无论是否有使用)，第一次携带，无公司机密资料，被查出者（）
```
A： 记小过一次
B： 记大过一次
C： 开除
正确答案：B
```

9. 公司原则上执行标准工时制，员工正常工作时间：每天工作8小时，每周工作5天，共计40小时。每周至少休息（）天。
```
A： 1天
B： 2天
C： 3天
正确答案：A
```
